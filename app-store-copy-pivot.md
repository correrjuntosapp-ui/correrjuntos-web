## 📋 Pasos para aplicar

### App Store Connect
1. https://appstoreconnect.apple.com/apps/6758505910
2. Sidebar → Información de la versión (de la versión actual o nueva)
3. Pegar:
   - Subtitle
   - Promotional Text
   - Description
   - Keywords
   - What's New
4. Hacer lo mismo en español + inglés
5. Save

### Google Play Console
1. https://play.google.com/console/u/0/developers/6979904302857989185/app/4974038233158114800
2. Sidebar → Crecimiento → Presencia en la Tienda → Listing principal
3. Pegar:
   - Short description
   - Full description
4. Hacer lo mismo para cada idioma (ES + EN)
5. Save → Publicar (puede tardar 1-7 días en revisión)
